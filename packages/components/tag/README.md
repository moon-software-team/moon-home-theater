# @moon/tag

Content blocks help users quickly access information or take action. Choose from various layouts to enhance clarity and engagement.

Please refer to the [documentation](./docs) for more information.

## Installation

```sh
yarn add @moon/tag
# or
npm i @moon/tag
```
